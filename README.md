## Descripción
Project API REST Pokedex with Node, NestJS, Docker and Mongo

# Ejecutar en desarrollo

## Clonar el repositorio

## Installation - node_modules

```bash
$ npm install
```

## Instalar Nest CLI 

```bash
# Global
$ npm i -g nestjs/cli

```

## Levantar la base de datos

```bash
# development
$ docker-compose up -d
```

## Ejecutar la aplicación

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
